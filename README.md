# unity-tskit



这是一个unity项目脚手架，采用的技术方案如下：

* Unity 2019/2020 lts
* TypeScript 脚本(puerts)
* FairlyGUI 界面



提供以下特性：

* 快速开始一个项目
* 合理的目录划分，适用于高效工作流程






目录结构：

* 工程 - 主要面向开发，建议使用git进行版本管理
  * UnityProject 主项目
  * TsProject 脚本项目
  * UIProject 界面项目
* 资源 - 主要面向策划和美术，使用svn进行版本管理
  * UI资源
  * 策划文档
  * 创意
  * 故事文案
  * 美术资源
  * 音乐音效
